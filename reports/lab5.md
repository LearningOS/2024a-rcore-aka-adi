## 实现
1，分别独立对mutex 和 semaphore实现银行家算法进行死锁检测。为了编码方便，检测采用遍历完成。
## 简答

### 1,
```
(1)回收进程空间等等资源
(2)需要回收系统的任务调度队列和等待队列，系统调度队列需要回收，等待队列不需要，因为不会再有唤醒函数的调用
```
### 2,
```
(1) 会解锁后唤醒一个任务，此时该任务并没有得到锁所有权确仍然进入了临界区
(2) 会唤醒一个等待任务后，不释放锁，即锁现在由刚唤醒的task负责
```
